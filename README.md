Verizon open source homepage

### Local Development

In order to build this project, follow these steps:

```
$ npm install
$ npm run build
```

In order to serve the site locally, you need to ensure you have installed the following:

```
$ npm install -g serve
```

Then serving the site is as easy as:

```
$ cd target
$ serve
```
